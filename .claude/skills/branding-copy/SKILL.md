# Branding & Copy — Evolutto / Funil Mascarado

## A marca: Evolutto

**O que é:** Sistema operacional para consultorias. Não é ferramenta, não é SaaS genérico — é uma plataforma pensada especificamente para a realidade de quem vende conhecimento.

**Posicionamento:** A Evolutto resolve o problema que ferramentas genéricas não conseguem: dar estrutura operacional a negócios de consultoria para que o CEO pare de ser o gargalo.

**Narrativa central:** "O problema nunca foi falta de esforço. Foi falta de um sistema feito para consultoria."

## A personagem: Eva

Eva é a voz do funil. Ela se comunica como se estivesse revelando algo secreto e urgente ao lead. Não é formal, não é corporativa — é direta, intimista e levemente dramática.

### Traços de personalidade da Eva
- **Direta** — frases curtas, sem rodeios
- **Intimista** — fala como se conhecesse o lead pessoalmente
- **Urgente** — sempre há uma tensão, um "antes que descubram"
- **Empática** — reconhece a dor do lead sem julgamento
- **Provocativa** — faz perguntas que incomodam (no bom sentido)

### Como Eva fala

**No WhatsApp** (mensagens curtas, uma ideia por mensagem):
```
ok
agora dá pra falar com mais calma
se você chegou até aqui
é porque alguma coisa encaixou
```

**Na chamada** (frases com timing, ritmo de conversa real):
```
escuta...
sou eu
Eva
do Evolutto
não tenho muito tempo
eles descobriram que eu revelei o protocolo
```

**Na Sales Page** (mais elaborada, mas ainda direta):
```
"Minha consultoria é complexa demais."
→ É exatamente para isso que a Evolutto existe.
```

### Regras de copy da Eva

1. **Nunca use jargão corporativo** — nada de "sinergia", "escalabilidade", "ROI"
2. **Frases curtas** — média de 5-10 palavras por mensagem no WhatsApp
3. **Minúsculas** — Eva não usa maiúsculas no WhatsApp (exceto nomes próprios)
4. **Sem pontuação excessiva** — um ponto final ou reticências, nunca exclamação
5. **Primeira pessoa** — "eu vou te mostrar", não "nós vamos apresentar"
6. **Tom de segredo** — como se estivesse compartilhando algo exclusivo

## Argumentos de venda (em ordem de aparição no funil)

### 1. O diagnóstico (Chamada + WhatsApp)
- Consultorias usam ferramentas que não foram feitas para elas
- CRM, Notion, Slack, planilhas — cada uma resolve um pedaço
- Nenhuma resolve o "todo"
- O CEO vira o gargalo sem perceber

### 2. A dor (WhatsApp + TikTok)
- "Se você sumisse por 30 dias, a empresa seguiria?"
- Trabalha mais e anda menos
- Mais controle aparente, menos controle real
- Não é desorganização — é ausência de sistema

### 3. A solução (TikTok + Sales)
- Evolutto é um sistema operacional, não uma ferramenta
- Pensado desde o início para consultoria
- Decisões não ficam presas no CEO
- Processos deixam de ser implícitos
- A empresa começa a andar sozinha

### 4. Objeções e respostas (Sales Page)

| Objeção | Resposta |
|---------|---------|
| "Minha consultoria é complexa demais" | É exatamente para isso que a Evolutto existe |
| "Já tentei outras ferramentas" | Ferramentas não são sistema operacional |
| "Não tenho tempo para implementar" | Você já está pagando esse custo todos os dias |
| "Isso substitui minha equipe?" | Não. Torna a equipe funcional sem você no centro |

## CTAs do funil

Os CTAs seguem uma escala de comprometimento crescente:

1. **Aceitar chamada** (baixo) → "Aceitar" (botão verde)
2. **Confirmar confiança** (médio) → "pode confiar" (botão WhatsApp)
3. **Escolher continuar** (médio) → "Ok, mostra." (quick reply)
4. **Abrir TikTok** (alto) → "Abrir TikTok privado" (botão preto)
5. **Acessar sistema** (alto) → "ACESSAR SISTEMA" (botão hacker verde)
6. **Comprar** (máximo) → CTA final na SalesPage

## Cores e significado

- **Verde** = ação positiva, avanço, confiança (aceitar chamada, WhatsApp, hacker)
- **Vermelho** = recusar, desligar (apenas botão decline)
- **Preto** = mistério, exclusividade (TikTok, hacker)
- **Emerald/dark** = premium, sofisticação (Sales Page)
